# Rumble Boxing - Assets Guide

## Required Files

For the website to display correctly, please ensure the following files are present in the `/public` directory:

### 1. Video Background

- **Filename**: `boxing-background.mp4`
- **Purpose**: Hero section background video
- **Requirements**:
  - Format: MP4
  - Duration: 20-30 seconds (for smooth looping)
  - Content: Boxing-themed footage (training, shadowboxing, etc.)
  - No audio needed (the video plays muted)
  - Compressed for web (recommend 720p or 1080p)

### 2. SVG Patterns

- **Filename**: `boxing-pattern.svg`
- **Purpose**: Subtle pattern overlay in hero section
- **Current Status**: Basic placeholder created

- **Filename**: `glove-outline.svg`
- **Purpose**: Boxing glove silhouette in hero section
- **Current Status**: Basic placeholder created

## Customization

Feel free to replace the placeholder SVGs with more detailed and on-brand designs. The current files are basic outlines that demonstrate the positioning and purpose.

## Where to Find Media

- **Videos**: Stock video sites (Pexels, Shutterstock, etc.), your own recorded content, or licensed footage
- **SVGs**: Create custom designs in Illustrator/Figma or find boxing-themed vector graphics on sites like Freepik or Vecteezy

Remember to ensure you have the rights to use any media you include in your project.
